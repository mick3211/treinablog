Simple blog made with NextJS

## Getting Started

Clone repository: ```git clone https://github.com/mick3211/treinablog.git```

Install dependencies: ```npm install```

Create a file in the root directory named ``.env.local`` and inside it: ```NEXT_SECRET = SOMERANDONSTRING```. This is used by NextAuth for authentication.

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
